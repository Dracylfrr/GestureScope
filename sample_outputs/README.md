# Sample Outputs

After running the app, exported reports will appear in `outputs/session_<timestamp>/`.

Expected files:

- `session_events.csv` — spreadsheet-friendly frame-by-frame gesture observations.
- `session_events.json` — full structured event log, including landmarks.
- `session_summary.json` — summary statistics.
- `gesture_report.md` — classroom-friendly report document.
- `gesture_report.html` — easy-to-open browser report.
